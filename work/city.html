City Tracker 1.0

This a search engine for different cities. It uses a series of API's including Zomato, Openweather API, Google Maps, and the Wikipedia Api.

This is an application I've been working on my free time. I'm getting familiarized with service pages, call backs, and parsing API data. I am hoping to add features including the Eventbrite API and a News API to see what is happening in each city and any news about that area.

Try it out though, type in where you are from and it will tell you things! Enjoy!

    How to use:
  1. If the city you type doesn't work on any of the API's: Wiki API, etc. try adding country or state i.e. "Portland, Or"
  2. Zomato is a free API but only has restaurant recommendations for certain countries (check here for more details
  3. Enjoy! Try any city in the world! One of my goals for this app is to learn about new places!

Check out the specs and ReadMe: https://github.com/jessecallahan/city-tracker
